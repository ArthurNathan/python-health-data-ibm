# Módulo 2 – Estruturas de Dados e Lógica

## 🧠 Objetivo
Praticar a organização e manipulação de dados em Python, utilizando listas, dicionários, loops e condicionais, aplicando em um contexto simples de dados clínicos.

---

## 📚 Conteúdo aprendido
- Listas
- Dicionários
- Acesso a dados (índice e chave)
- Loop `for`
- Condicional `if`

---

## 💡 Projeto
Simulação de um pequeno conjunto de dados de pacientes, com informações como nome, idade, altura e condições clínicas.

---

## ⚙️ O que o código faz
- Acessa dados específicos de pacientes
- Percorre todos os registros
- Exibe informações selecionadas
- Filtra pacientes com hipertensão
- Conta quantos pacientes possuem diabetes

---

## 🧩 Aprendizados principais
- Diferença entre acessar lista e dicionário
- Entendimento de erros comuns (`TypeError`, acesso inválido)
- Uso de loops para percorrer dados
- Uso de condicionais para filtrar informações
