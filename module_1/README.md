# Módulo 1 – Manipulação de Strings

## 🧠 Objetivo
Praticar a manipulação de strings em Python, aplicando o conhecimento em um contexto clínico simples, com foco na adaptação e organização de dados.

---

## 📚 Conteúdo aprendido
- Strings
- Métodos de string
- Manipulação de texto

---

## 💡 Projeto
Simulação de anonimização de dados de pacientes, substituindo os nomes por suas iniciais como forma simples de preservação de privacidade.

---

## ⚙️ O que o código faz
- Recebe nomes de pacientes
- Processa os nomes
- Converte para iniciais
- Gera uma forma simplificada de anonimização

---

## 🧩 Aprendizados principais
- Manipulação básica de strings
- Aplicação prática em contexto de saúde
- Importância da organização e tratamento de dados
