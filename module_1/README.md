# Módulo 1

A ideia deste mini projeto é registrar o aprendizado presente no módulo 1, ao mesmo tempo em que me proponho a aplicá-lo. Para isto, pretendo listar o conteúdo aprendido e, em seguida, escrever um código aplicando num cenário clínico/saúde possível. 

Meu objetivo com isso é integrar o conhecimento adquirido, incialmente básico, com a prática de elaborar um projeto aplicando-o ao contexto da Saúde.

## Projeto
